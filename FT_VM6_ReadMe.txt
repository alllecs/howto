I. Установка библиотек и зависимостей:
 
 1.	Перейти в корневую директорию программы, выполнив команду:
		cd /%path_to_program%/
 2.	Для 32-битных ОС выполнить команду:
		sh install_32bit.sh
	
	Данная комманда запускает скрипт, распаковывающий и копирующий файлы необходимых 32-битных библиотек,
	проверяет их успешную установку и копирует Makefile для 32-битных ОС.
	В случае успешной установки вывод в консоль (CHECKING INSTALLED LIBRARIES) покажет информацию обо всех
	присутствующих файлах нужных библиотек.


 3. Для сборки программы необходимо последовательно выполнить следующие команды:

 	3.1. make clean	- производит удаление всех устаревших исполняемых и объектных файлов

 	3.2. make - производит сборку программы, используя распакованный ранее Makefile.

II.		Подготовка к использованию:
 
 1. Вывод списка подключенных устройств и интерфейсов: подключить устройство, в терминале ввести команду "ls -l /dev/ttyUSB" и нажать TAB. На экран будет выведен список ttyUSB-устройств (USB <-> Serial converter). Как правило, необходмый интерфейс I/O - ttyUSB1 для первого устройства, ttyUSB3 - для второго ит.д.

 2. Вызов терминала. Выполнить команду "sudo minicom -s" (если по умолчанию нет root прав, то без них minicom будет недоступен).



III.	Использование программы:
 
 1. Вызов справки по использованию программы:
 		./ft_control -h 
 		или 
 		./ft_control -?
 
 2.	Вывод подключенных USB-устройств. Отобразит все USB-устройства в формате Bus:Port:DeviceID:Interface:SerialNumber:Description.
 		./ft_control -i

 3. Выбор устройства для работы. В случае, если в системы более 1 FTDI-устрйоства, программа потребует указать для какого устройства выполнять 		функции. Выбор устрйоства осуществляется в формате Bus:DeviceID, принимается как формат с нулями,так и без (например, 003:001 или 3:1):

 		./ft_control -d <bus>:<devID> -[прочие опции]

 		Пример: ./ft_control -d 003:005 -P -v h -s nmi_n - позволит включить устройство на шине 003 с ID 005 с указанными параметрами. 

 3.	Включение устройства:

 	3.1. Включение с номинальным напряжением, опции служебных сигналов по умолчанию:
 		./ft_control -P
 		или 
 		./ft_control -P -v n

 	3.2. Включение с повышенным напряжением, опции служебных сигналов по умолчанию:
 		/ft_control -P -v h

 	3.3. Включение с пониженным напряжением, опции служебных сигналов по умолчанию:
 		./ft_control -P -v l

 	3.4. Включение с любым напряжением и выбором опций служебных сигналов:
 		./ft_control -P -v [n,h,l] -s [nmi_n]

 		!Необходимо указать один из вариантов аргументов для каждой из опций с аргументами!

 4. Сигнал системного сброса:

 		./ft_control -r

 		Осуществляет сброс и перезагрузку устройства. После сброса выставляется номинальное напряжение.

 5. Выключение устройства:

 		./ft_control -p

 		Осуществляет выключение устройства и сброс установленных параметров.
